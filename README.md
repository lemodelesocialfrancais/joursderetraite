## Français

**Jours de Retraite** est un outil web qui permet de visualiser le temps de retraite représenté par un montant donné, en années, mois et jours.

### Fonctionnalités

- Calcul instantané du temps de retraite
- Interface adaptée aux mobiles (PWA)
- Thème sombre élégant
- Mode hors-ligne

### Technologies

HTML · CSS · JavaScript · PWA

### Démonstration

🔗 [Voir la démo](https://lemodelesocialfrancais.github.io/joursderetraite/)
